# Data Manipulation Using NumPy & Pandas

This project demonstrates data cleaning, transformation, grouping, filtering, and 
statistical calculations using NumPy and Pandas on a Coffee Shop Sales dataset.

## Features:
✔ Load & clean dataset  
✔ Convert dates  
✔ Add new columns (Month, Day, Sales_Tag)  
✔ Group & aggregate data  
✔ Filter & sort  
✔ NumPy calculations (mean, min, max)  
✔ Prepare dataset for visualization  

The Jupyter Notebook includes each step with explanations and code.
